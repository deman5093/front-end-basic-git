  • git status - показує стан нашого репозиторію
  • git branch - які є гілки
  • git add . - додати поточну папку
  • git commit -m повідомлення що відбулося (що змінено)
  • git log - подивитися хто,що,коли робив
  • git push origin HEAD - HEAD-поточний репозиторій. Додати об'єкти,коміти
  • git checkout -b 'назва' - створити і перейти на нову гілку
  • git pull - отримати зміни
